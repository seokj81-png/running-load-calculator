# 러닝 강도·피로 계산기 (PAFGYM)

러닝 앱 기록 사진을 올리면 데이터를 읽어 운동 부담을 계산하고, 다음 7일 운동 계획을 제안하는 웹 도구입니다.

- **일반인 버전**: 사진 한 장 → 오늘 부담 + 7일 운동 계획
- **전문가 버전**: 5가지 부하 지표(체감 강도·심박 부하·페이스 점수·출력·강도 분포)로 직접 계산
- 사진 분석(OCR)은 **사용자 브라우저 안에서만** 실행 — 사진이 서버로 전송되지 않습니다.
- 단일 HTML 파일. 외부 의존성은 Tesseract.js(OCR) CDN 하나뿐.

대한혈관학회 2026 «Outdoor Running» 발표 PERFORMANCE 파트 기반.

> 이 도구는 훈련 부하 모니터링 보조용이며 의학적 진단·처방을 대체하지 않습니다.
